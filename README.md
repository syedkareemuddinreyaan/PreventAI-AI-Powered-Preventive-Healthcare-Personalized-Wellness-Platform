# PreventAI

PreventAI is an AI-powered preventive healthcare platform that helps users identify potential health risks and receive personalized wellness recommendations before serious conditions develop.

The platform combines machine learning, explainable AI, fitness planning, nutrition guidance, health simulation, and automated reporting into a unified healthcare experience.

## Problem Statement

Many chronic health conditions such as heart disease, diabetes, obesity, sleep disorders, and mental health issues are strongly influenced by lifestyle factors. Existing solutions often focus on monitoring health rather than preventing disease progression through actionable insights.

PreventAI addresses this challenge by providing risk prediction, personalized recommendations, and long-term health improvement planning within a single platform.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/PreventAI.git
cd PreventAI
```

Install Python dependencies:

```bash
pip install -r requirements.txt
```

Install frontend dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```
